🖼️ BG Preview Extension

Preview background images on real websites before sending to stakeholders.

⬇ Download Extension (ZIP)

Step 1 — Install the Extension

1

Open Google Chrome and type this in the address bar:
chrome://extensions/
Press Enter.

2

In the top right corner, turn on Developer mode (it's a small toggle switch).

3

Click the Load unpacked button that appears on the top left.

4

A file browser opens. Navigate to the folder where you unzipped this file, select the bg-preview-extension folder, and click Select Folder.

5

Done! The BG Preview extension now appears in your Chrome toolbar. You only need to do this once.

Step 2 — Use It

1

Open the website you want to preview the background on.

2

Click the BG Preview icon in your Chrome toolbar (top right — looks like a puzzle piece 🧩, then find BG Preview).

3

A small panel opens. Click or drag your image into the upload area.

4

Select a display mode (see below), then click Apply to Page.
The background on the website changes instantly — only you can see it.

5

Take your screenshots. When done, click Remove Preview to go back to normal.

Device Preview — Desktop or Mobile?

Desktop ✓ Default

Applies the background to the full page at 1920 × 1080. Use this for all standard desktop reviews.

Mobile

Shows a floating 856 × 475 preview frame over the page. Click ✕ or anywhere outside to close it.

Display Modes — Which One to Use?

Fixed ✓ Recommended

Fills the screen and stays still while scrolling. Most accurate for this website.

Cover

Fills the full page. Can look stretched on very long pages.

Contain

Shows the full image without cropping. May leave gaps on the sides.

Repeat

Tiles the image like a pattern.

Important: This extension only changes how the page looks on your screen. It does not edit the real website. Nothing is saved, nothing is broken. Refreshing the page removes the preview automatically.
